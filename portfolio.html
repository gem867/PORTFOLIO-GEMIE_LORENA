Visit Subway Surfers
Description:

"Subway Surfers" is an endless runner game where players control a character running through subway tracks, avoiding obstacles like trains and barriers, while collecting coins and power-ups. The game features vibrant, colorful graphics and smooth animations, offering an engaging experience that gets more challenging as the player progresses. With randomized environments and changing objectives, the game keeps players hooked with every run.

Role in the Project:

I worked as a game designer and developer, focusing on creating levels, designing obstacles, and fine-tuning the game's difficulty curve. Additionally, I optimized the game for both mobile and desktop platforms, ensuring it runs efficiently across different devices.

Tools/Technologies Used:

Unity, C#, Firebase (for leaderboards), Adobe Photoshop, GitHub

Description:

A comprehensive video streaming platform similar to Netflix that allows users to browse a catalog of movies and TV shows, receive personalized recommendations, and play HD video content. The app includes features such as genre sorting, search functionality, and user profile management, providing a smooth and immersive viewing experience across devices.
